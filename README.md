# 智学闭环

基于 AI 的学习资料智能测评与个性化复习工具。项目将“资料输入、知识点提取、智能出题、在线测评、错因诊断、复习计划、强化练习、学习报告”串成完整学习闭环，适合黑客松和比赛路演展示。

第二版重点增强了“可换资料演示”能力：系统会根据用户输入的学习资料半动态提取知识点并生成题目，不再只能演示固定的人工智能示例内容。

## 核心功能

- 资料输入：支持粘贴文本、上传 `.txt` 文件，并内置《人工智能基础概念》示例资料。
- 知识点提取：模拟 AI 从学习资料中提取知识点、重要程度、掌握目标和考查方式。
- 智能出题：根据知识点动态生成单选题、判断题和简答题。
- 在线测评：支持真实交互答题、未答完提交确认、一键优秀答卷和一键含错答卷。
- 错因诊断：根据错题生成概念混淆、关键词遗漏、场景判断错误等诊断。
- 个性化复习：生成 3 天复习计划和薄弱知识点强化练习。
- 教师端预览：展示上传课件、生成随堂测、班级掌握率、薄弱排行和分层任务。
- 报告导出：生成 Markdown 学习报告，支持复制、下载和浏览器打印 / 导出 PDF。

## 第二版新增功能

- 半动态 Mock AI：`src/services/aiService.ts` 会拆分段落和句子，识别“是、包括、分为、特点、作用、原因、影响、应用、区别、流程、原则”等关键词，自动生成 4-8 个知识点。
- 动态题目生成：题目围绕当前资料的知识点生成，不再固定为 AI 基础概念题。
- 路演答卷按钮：答题页提供“一键生成优秀答卷”和“一键生成含错答卷”，便于快速展示高分报告或错因诊断链路。
- 错题证据增强：结果页展示用户答案、正确答案、答案解析、对应知识点和得分情况。
- 首页路演表达增强：增加学习痛点、解决方案、应用场景和教师端扩展预览。
- 步骤条可回看：已完成步骤可以点击返回，未完成步骤保持不可点击。

## Demo 演示流程

1. 打开首页，点击“开始体验”。
2. 点击“使用示例资料”。
3. 点击“开始分析”，查看 AI 提取的知识点。
4. 点击“生成测评题目”，查看题目列表。
5. 点击“开始答题”，完成单选、判断、简答题。
6. 点击“提交测评”，查看总分、掌握率和薄弱知识点。
7. 点击“生成错因诊断”。
8. 点击“生成个性化复习计划”。
9. 点击“生成强化练习”，查看薄弱点强化题。
10. 点击“生成学习报告”，复制或下载 Markdown 报告。

## 一键演示流程

1. 打开首页，先讲“学习痛点、解决方案、应用场景、教师端扩展”。
2. 点击“开始体验”，使用示例资料或粘贴一段新的课程资料。
3. 点击“开始分析”，强调系统会根据资料半动态提取知识点。
4. 点击“生成测评题目”和“开始答题”。
5. 点击“一键生成含错答卷”，再提交测评。
6. 在结果页展示错题证据：用户答案、正确答案、解析、知识点、得分。
7. 继续生成错因诊断、复习计划、强化练习和学习报告。
8. 在报告页点击“打印 / 导出 PDF”，展示可交付报告。

## 路演推荐操作顺序

推荐用“先价值、再闭环、最后扩展”的顺序：

1. 首页：先讲学生复习痛点，再讲资料驱动闭环。
2. 资料输入：展示可以替换成任意课程资料。
3. 知识点页：强调 4-8 个知识点来自材料，而不是固定题库。
4. 答题页：使用“一键生成含错答卷”，快速制造诊断样本。
5. 结果页：展示可解释评分依据。
6. 诊断和计划：强调错题不是终点，而是复习路径的起点。
7. 教师端预览：说明可扩展为班级学情分析和分层作业平台。

## 项目亮点

- 从资料到测评的完整学习闭环
- 基于错题的薄弱知识点诊断

## 项目答辩亮点

- 可替换资料：评委可以现场输入新材料，系统仍能生成知识点和题目。
- 可解释测评：错题列表保留用户答案、标准答案、解析和得分，不是黑盒评分。
- 闭环完整：从资料到诊断再到复习计划和强化练习，形成可落地学习路径。
- 产品可扩展：教师端预览展示班级掌握率、薄弱排行和分层任务，具备平台化想象空间。
- 技术可迁移：当前使用稳定 mock，后续可替换为真实 LLM API，不影响前端流程。

## 环境变量

复制 `.env.example` 为 `.env` 后按需配置：

```bash
VITE_AI_PROVIDER=mock
VITE_OPENAI_API_KEY=
VITE_DEEPSEEK_API_KEY=
VITE_QWEN_API_KEY=
```

默认 `VITE_AI_PROVIDER=mock`，不需要后端和数据库即可运行完整 Demo。

## 接入真实大模型 API 的方式

真实 API 入口保留在 `src/services/aiService.ts`：

1. 修改 `.env`：

```bash
VITE_AI_PROVIDER=openai
VITE_OPENAI_API_KEY=你的 Key
```

2. 在 `realAiPlaceholder` 附近按 Provider 封装请求。
3. 让模型返回结构化 JSON，字段保持与 `KnowledgePoint`、`QuizQuestion`、`DiagnosisItem` 等类型一致。
4. 保留 mock 作为离线演示 fallback，避免比赛现场网络不稳定影响演示。

## 比赛路演介绍话术

“智学闭环”不是单点式 AI 出题工具，而是面向学生自主复习的完整闭环系统。用户只需要上传自己的学习资料，系统就能自动提取知识点、生成测评题目、完成在线评分，并根据错题定位薄弱知识点，进一步生成复习计划和二次强化练习。它把传统复习中分散的资料整理、刷题、批改、错题复盘和计划制定整合到一个轻量化 Web 工具中，既适合学生个人使用，也可以扩展到教师端进行班级学情分析。
