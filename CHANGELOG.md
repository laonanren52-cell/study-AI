- 将资料上传页优化为多格式文件导入中心，突出 TXT / PDF / DOCX / PPTX 本地解析能力。
- 优化题目预览、在线答题、测评结果、错因诊断、复习计划、强化练习和报告导出页面的信息层级。

- 新增打印样式，报告导出页更接近正式报告预览界面。

## 2026-05-24 第四版优化

- 重写 `aiService.ts` 的知识点提取和 Mock 出题逻辑，避免碎片词题干和碎片选项。
- 为知识点新增来源依据和关键词，为题目新增来源依据和质量评分。

- 新增 `src/components/AIStatusBadge.tsx`，在页面显示 Mock/API 当前状态。
- 题目页展示来源依据和题目质量评分。
- 更新 `.env.example` 和 README，补充 API 配置方式、生产安全提醒和题目质量标准。

## 2026-05-24 第三版优化

- 新增 `src/services/fileParser.ts`，支持 `.txt`、`.pdf`、`.docx`、`.pptx` 本地解析。
- 新增 `src/utils/textClean.ts`，提供文本清洗、字数统计和文件大小格式化。
- 安装 `pdfjs-dist`、`mammoth`、`jszip`，分别用于 PDF、Word、PPTX 解析。
- 资料输入页支持点击上传、拖拽上传、解析中状态、解析失败提示和文件元信息展示。
- 报告中新增学习资料来源信息，包括文件名、文件类型、文件大小、提取字数和页数。
- 优化半动态知识点提取，过滤“第 X 页”“目录”“谢谢观看”等低价值内容。
- 更新 README，补充第三版多格式资料上传能力、限制说明和路演流程。

## 2026-05-24 第二版优化

- 答题页支持未答完提交确认，新增“一键生成优秀答卷”和“一键生成含错答卷”。
- 结果页错题列表新增用户答案、正确答案、答案解析、对应知识点和得分情况。
- 首页新增学习痛点、解决方案、应用场景和教师端扩展预览。
- 新增 `TeacherDashboardPreview.tsx`，展示教师端课件上传、随堂测、班级掌握率、薄弱排行和分层任务。
- 报告页新增浏览器打印 / 导出 PDF 功能。
- 步骤条支持点击已完成步骤返回回看。
- 新增 `.env.example`，明确 mock 模式和真实大模型 API 预留配置。
- 更新 README，补充第二版功能、一键演示流程、路演顺序和真实 API 接入方式。
